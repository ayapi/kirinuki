# Requirements Document

## Introduction

チャンネルIDの指定が必要なCLIコマンド（`channel videos`、`suggest`）において、登録チャンネルが1つだけの場合にそのチャンネルIDを自動的にデフォルトとして使用し、明示的な指定を省略可能にする機能の要件を定義する。
現在、対象コマンドではチャンネルIDが必須の位置引数となっており、登録チャンネルが1つでも毎回指定が必要である。

## Requirements

### Requirement 1: チャンネルID省略時の自動解決

**Objective:** ユーザーとして、登録チャンネルが1つだけの場合にチャンネルIDの指定を省略してコマンドを実行したい。これにより、日常的な操作のタイプ量を削減し、ワークフローを簡略化できる。

#### Acceptance Criteria

1. When チャンネルIDが省略された状態でコマンドが実行され、登録チャンネルが1つだけ存在する場合, the CLI shall その唯一の登録チャンネルIDを自動的に使用してコマンドを実行する
2. When チャンネルIDが省略された状態でコマンドが実行され、登録チャンネルが複数存在する場合, the CLI shall チャンネルIDの指定が必要である旨のエラーメッセージと登録チャンネル一覧を表示する
3. When チャンネルIDが省略された状態でコマンドが実行され、登録チャンネルが0件の場合, the CLI shall チャンネルが未登録である旨のエラーメッセージを表示する
4. When チャンネルIDが明示的に指定された場合, the CLI shall 登録チャンネル数に関わらず、指定されたチャンネルIDをそのまま使用する

### Requirement 2: 対象コマンドへの適用

**Objective:** ユーザーとして、チャンネルIDを必要とするすべてのコマンドで一貫してデフォルトチャンネル機能を利用したい。これにより、コマンド間で異なる動作に混乱することなく操作できる。

#### Acceptance Criteria

1. The CLI shall `channel videos` コマンドでチャンネルID省略時の自動解決をサポートする
2. The CLI shall `suggest` コマンドでチャンネルID省略時の自動解決をサポートする
3. While チャンネルIDが省略された場合, the CLI shall すべての対象コマンドで同一のデフォルト解決ロジックを使用する

### Requirement 3: ユーザーフィードバック

**Objective:** ユーザーとして、デフォルトチャンネルが自動選択された場合にどのチャンネルが使用されたかを把握したい。これにより、意図しないチャンネルに対する操作を防止できる。

#### Acceptance Criteria

1. When デフォルトチャンネルが自動選択された場合, the CLI shall 使用されるチャンネル名とチャンネルIDを標準エラー出力に通知メッセージとして表示する
2. When 複数チャンネルが登録されておりチャンネルIDが省略された場合, the CLI shall エラーメッセージに登録チャンネルの一覧（チャンネル名とID）を含める
