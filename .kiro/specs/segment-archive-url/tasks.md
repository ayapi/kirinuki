# Implementation Plan

- [x] 1. YouTube URL生成の共通関数を作成しテストする
  - `video_id` と開始ミリ秒を受け取り、`https://www.youtube.com/watch?v={video_id}&t={秒}` 形式のURLを返す関数を `clip_utils` に追加する
  - ミリ秒→秒変換は整数除算で小数点以下を切り捨てる
  - 正常値、0ms、大きな値、切り捨て確認（例: 1999ms → 1s）のテストケースを書く
  - _Requirements: 2.2, 2.3_

- [x] 2. 各コマンドで共通関数を使用しURL表示を統一する
- [x] 2.1 (P) `segments` コマンドの出力に各セグメントのYouTube URLを追加する
  - 各セグメントの時間範囲・要約行の下に、該当区間から視聴できるURLを表示する
  - URLの `&t=` パラメータにはセグメント開始時刻の秒数（切り捨て）を設定する
  - セグメントが0件の場合は既存メッセージのみ表示しURLは出力しない
  - コマンド出力にURL行が含まれること、0件時に含まれないことをテストで検証する
  - _Requirements: 1.1, 1.2, 1.3_

- [x] 2.2 (P) `search` コマンドのURL生成を共通関数に切り替える
  - `SearchService` 内のプライベートなURL生成メソッドを削除し、共通関数を呼び出すよう変更する
  - 検索結果のURL表示が従来と同じ形式であることを既存テストで検証する
  - _Requirements: 2.1, 3.1_

- [x] 2.3 (P) `suggest` コマンドのURL生成を共通関数に切り替える
  - `RecommendationFormatter` の静的URL生成メソッドを削除し、共通関数を呼び出すよう変更する
  - 推薦結果の秒単位の時刻をミリ秒に変換して共通関数に渡す
  - テキスト出力の `URL:` 行とJSON出力の `youtube_url` フィールドが従来と同一であることをテストで検証する
  - _Requirements: 2.1, 3.2, 3.3_
