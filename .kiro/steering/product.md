# Product Overview

YouTube Liveの配信アーカイブを「知識ベース」として活用するCLIツール。
字幕データの解析により話題区間を自動抽出・推薦し、複数動画を横断して「あの話どこでしてた？」を検索できる。
もちろん、指定区間の切り抜き動画生成にも対応。

## Core Capabilities

1. **字幕・メタデータ取得** — YouTube Liveアーカイブ（メンバー限定含む）から字幕とメタデータのみを取得・蓄積（動画本体はDLしない）
2. **字幕解析・話題抽出** — 字幕データから話題の区間を自動セグメント化し、切り抜き候補として推薦
3. **横断検索** — 「このテーマで話していた箇所」を複数動画にまたがって検索
4. **オンデマンドクリッピング** — 切り抜き実行時にはじめて動画を取得し、指定区間のみ切り出して保存。元動画は即時削除
5. **メタデータ・インデックス管理** — 動画情報と字幕インデックスの永続的な管理

## Target Use Cases

- 「前にこの話してたけど、どの配信のどこ？」をすぐ見つけたい
- 長時間配信から切り抜きに使えそうな話題を自動で提案してほしい
- メンバー限定配信を含む複数アーカイブを横断的に検索・管理したい
- 見つけた区間をそのまま切り抜き動画として書き出したい

## Value Proposition

- **字幕ベースのインテリジェンス**: LLMによる話題セグメンテーションと意味検索
- **メンバー限定対応**: Cookie認証で限定動画の字幕・動画にアクセス
- **ストレージ効率**: 字幕・メタデータのみローカル蓄積。動画は切り抜き時にオンデマンド取得し、完了後即削除
- **ローカル完結**: インデックスはローカルDB、動画処理もローカル
- **CLIファースト**: 自動化・スクリプト連携が容易
